I made this to learn about the Django Web Framework.

This is a simple Kanband Board App.

This features the following:

- Homepage Dashboard
- Create / Update / Delete Tasks
- Create / Update / Delete Tasks Comments
- Task Related Views
- User Login / Logout
- User Specific Task Page
